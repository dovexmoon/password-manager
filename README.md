# Password Manager

A simple Password Manager built with Python that allows users to securely store and retrieve account credentials through a menu-driven interface.

## Features

* Add new website credentials
* View saved credentials
* Generate strong random passwords
* Store data using JSON
* Easy-to-use terminal interface
* Beginner-friendly Python project

## Technologies Used

* Python
* JSON
* Random Module
* String Module

## Project Structure

```text
Password-Manager
│
├── password_manager.py
├── passwords.json
├── README.md
└── .gitignore
```

## How It Works

1. User selects an option from the menu.
2. Credentials are stored in a JSON database.
3. Stored credentials can be retrieved using the website name.
4. The built-in password generator creates strong random passwords.

## Example Menu

```text
PASSWORD MANAGER

1. Add Password
2. View Password
3. Generate Password
4. Exit
```
